# currentzeng.com

Portfolio scaffold for Current Zeng.

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Ground Rules

- Design tokens live in `src/config/design-tokens.ts`.
- Display strings live in `src/config/copy.ts`.
- Shared UI lives in `src/components`.
- Shared helpers live in `src/lib`.
- Source files should stay under 600 lines.

## Checks

```bash
npm run lint
npm run build
```
